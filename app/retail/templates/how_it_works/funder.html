{% blocktrans %} Decide on a project type, permissions and complete the Fund Issue form. {% endblocktrans %}

{% blocktrans %} Once work has been completed the contributor will submit a PR for review. {% endblocktrans %}

{% blocktrans %} Depending on the project type, payment can be made to one contributor or to multiple contributors at one time. {% endblocktrans %}
